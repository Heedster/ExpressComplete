# ExpressComplete
A sublime text plugin for Expressjs Autocompletions

## Help:
* aget : ```app.get("/", function(req,res) { } )```
* apost : ```app.post("/", function(req,res) { } )```
* aput : ```app.put("/", function(req,res) { } )```
* adel : ```app.delete("/", function(req,res) { } )```
* rs : ```res.send()```
* rw : ```res.write()```
* re : ```res.end()```
* rr : ```res.render('')```
* rd : ```res.redirect('')```
* rqpar : ```req.params[]```
* rqb  : ```req.body```
* rqsess : ```req.session```
